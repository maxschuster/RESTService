      This class represents the rest service.

      Controller can be registered in an instance of this class and this class seeks the right controller for the request to be processed.

      Unlike a 401 Unauthorized response, authenticating will make no difference. On servers where authentication is required, this commonly means that the provided credentials were successfully authenticated but that the credentials still do not grant the client permission to access the resource (e.g. a recognized user attempting to access restricted content).
